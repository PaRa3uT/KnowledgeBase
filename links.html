Awesome-Cheatsheets - collection of popular web programming language cheatsheet

Color Hunt - Инструмент для формирования цветовой палитры
Web Gradients - На Web Gradients собрано 180 линейных градиентов, которые можно использовать на сайте в качестве фона для контента. Нужно только скопировать и вставить кроссбраузерный код CSS3. Также есть PNG-версия каждого градиента и пакеты для Sketch и Photoshop

Visual Type Scale - Инструмент позволяет увидеть разницу между цветом шрифта, размером и значениями rem (размер шрифта корневого элемента).
CodeMyUI - Сайт со сниппетами веб-дизайна и UI-сниппетами, которые можно свободно использовать в проектах
DevDocs - Ресурс с документацией практически по каждой технологии, используемой в веб-разработке. Для удобства есть функция поиска. Любой файл можно скачать для чтения офлайн.
Bundlephobia - С помощью Bundlephobia можно проверить размер зависимостей до добавления их в проект.
